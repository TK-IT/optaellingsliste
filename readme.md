# Optaellingsliste

LaTeX dokument for optælling af spilledåse og øl/vand. Dokumentet
findes i mappen Justin Biber i KASS/INKA-skabet.

For at generere nye datoer kan `skema.pl` køres med `perl
skema.pl`. Kopier og sæt ind.

## Krav
* `perl` for at kører `skema.pl`
* LaTeX installation
* `tket` LaTeX pakken.
